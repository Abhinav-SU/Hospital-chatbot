### Recommended Screen Recording Software:
- **OBS Studio** (Free, Windows/Mac/Linux) - https://obsproject.com/
- **Loom** (Free, Browser-based) - https://loom.com/
- **ShareX** (Free, Windows) - https://getsharex.com/
- **QuickTime** (Mac built-in)
- **SimpleScreenRecorder** (Linux)

### Audio Recording Tips:
1. Use a good microphone (or headset mic)
2. Record in a quiet room
3. Speak clearly and at moderate pace
4. Use Audacity (free) if you need to edit audio separately

---

## üé® Visual Tips

1. **Use Full Screen** - Press F11 in browser for distraction-free recording
2. **Zoom Level** - Set browser zoom to 100% or 110% for readability
3. **Mouse Highlighting** - Enable mouse cursor highlighting in your recording software
4. **Slow Typing** - Type slower than normal so viewers can follow
5. **Wait for Responses** - Pause 2-3 seconds after each response before moving on
